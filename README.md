# Network Throughput

[![Creative Commons](https://flat.badgen.net/badge/license/CC-BY-NC-4.0/orange)](https://creativecommons.org/licenses/by-nc/4.0/)

A simple Network Throughput monitor for [Ãœbersicht](http://tracesof.net/uebersicht) using the colors from [Monokai Faded](https://dionmunk.com/projects/monokai-faded/). The widget takes all of your available network connections and adds their upload and download speeds together and gives you a readout.

![Screenshot](screenshots/screenshot.png)


## Installation

- Download the [repository](https://github.com/dionmunk/ubersicht-network-throughput/archive/master.zip) and extract it.
- Place the `network-throughput.widget` folder in your Ubersicht extension folder.
- Refresh Ubersicht.
